Legend:
  ░ = Black background
  ▒ = Orange raster bars
  ▓ = Purple raster bars  
  █ = Green raster bars
  
Features:
  🎃 Four animated jack-o'-lantern pumpkin sprites
  🌊 Sine wave horizontal movement (80 pixel amplitude)
  📊 Vertical wobble effect (20 pixel amplitude)
  🌈 Halloween-themed raster bars (orange/black/purple/green/red)
  💬 Spooky greetings in orange text
  
Technical:
  - 4 sprites @ 2x width for visibility
  - Stable raster IRQ at line 50
  - 256-byte sine wave tables
  - Color cycling animation
  - Real-time C64 rendering

Memory Usage:
  $0801-$0804 : BASIC starter
  $2000-$203f : Pumpkin sprite data
  $2100-$21ff : Horizontal sine data
  $2200-$22ff : Vertical sine data
  $2300-$23ff : Color table data
  $2400+      : Main program
